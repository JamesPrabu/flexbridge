Overview

FLEx Bridge supports sharing a full FieldWorks data set with a remote FieldWorks FLEx user.

Usage

FLEx Bridge is a stand alone program which can be used when a FieldWorks application is not running.

Downloads

FLEx Bridge can only be used with the xml data file (fwdata extension) of FieldWorks 7+.
