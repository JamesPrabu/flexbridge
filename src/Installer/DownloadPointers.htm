Overview

FLEx Bridge supports sharing a full FieldWorks data set with a remote FieldWorks FLEx user.

Usage

FLEx Bridge is a program which can be used when FLEx is running by launching FLEx Bridge from FLEx.

Downloads

FLEx Bridge can only be used with the xml data file (fwdata extension) of FieldWorks 7.3.x, or higher.
